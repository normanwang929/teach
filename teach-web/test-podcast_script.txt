接下来我们聊聊「什么是变量？」。[停顿]

变量是程序中用来存储数据的容器。在 Python 中，变量不需要声明类型，直接赋值即可：
- isstudent 存储了一个布尔值

接下来我们聊聊「基本数据类型」。[停顿]

- int — 整数，如 42, -7, 0
- float — 浮点数，如 3.14, -0.5
- str — 字符串，如 "hello", 'Python'
- bool — 布尔值，True 或 False
- list — 列表，如 1, 2, 3
- dict — 字典，如 {"name": "Alice", "age": 25}
