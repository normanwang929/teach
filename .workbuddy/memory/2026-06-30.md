## W1+W2 Teach 平台开发（2026-06-30）

### 已完成

**W1 前端（teach-web/）**
- `index.html` — 课程列表首页，搜索+分类+排序
- `assets/styles.css` — 深色主题样式
- `assets/app.js` — 前端逻辑（进度追踪、筛选）
- `data/courses.json` — 课程索引（含演示数据）
- CloudStudio 部署预览可用

**W2 资料搜索引擎（scripts/）**
- `search_course_materials.py` — 主入口，CLI: `python search_course_materials.py "主题"`
- `quality_scorer.py` — 5维评分（来源/时效性/免费性/互动性/质量信号），总分0-100
- `dedup.py` — URL归一化 + 标题Jaccard相似度去重
- `sources/arxiv.py` — arXiv API，**可用**
- `sources/wikipedia.py` — MediaWiki API，**可用**
- `sources/bilibili.py` — 需配置 Cookie
- `sources/zhihu.py` — 需配置 Cookie
- `sources/youtube.py` — 需 YouTube API Key
- `sources/open_courses.py` — 部分实现（MIT OCW RSS）
- `sources/podcasts.py` — 教育播客 RSS，可用
- `sources/anysearch.py` — 待 AnySearch 部署后接入
